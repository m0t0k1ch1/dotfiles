## Install dependencies

### gocode

``` sh
$ go get -u github.com/nsf/gocode
```

### goimports

``` sh
$ go get -u golang.org/x/tools/cmd/goimports
```

### irony-server

``` sh
$ brew install llvm
```

```
M-x irony-install-server RET
```

## Install packages

``` sh
$ cask install
```

## Update packages

``` sh
$ cask update
```
